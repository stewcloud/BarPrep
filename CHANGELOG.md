\# Changelog



All notable changes to BarPrep will be documented in this file.



The format is based on Keep a Changelog:

https://keepachangelog.com/en/1.1.0/



\---



\# \[4.4] - 2026-05-25



\## Added

\- Cross-platform QR scanner page

\- Camera-based label scanning workflow

\- Manual QR/code lookup fallback

\- Placeholder BarPrep branding/logo

\- Scan Label navigation shortcut

\- Recent batch/service quick access on scanner page



\## Changed

\- Replaced browser-native BarcodeDetector implementation

\- Scanner now uses html5-qrcode library

\- Improved scanner messaging and HTTPS guidance

\- Updated dashboard layout for faster operations



\## Fixed

\- Improved iOS Safari QR scanner compatibility

\- Improved Windows Edge QR scanner compatibility

\- Improved mobile camera handling



\## Known Issues

\- Some older iOS devices may still require manual lookup

\- Emoji rendering depends on server/printer font support
